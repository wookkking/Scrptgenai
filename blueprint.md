# Blueprint: ScriptGen AI (릴스/쇼츠 스크립트 생성기)

## 1. 프로젝트 개요

- **목표:** 사용자가 입력한 주제(키워드)를 기반으로 인공지능을 활용하여 숏폼(릴스, 쇼츠, 틱톡) 비디오용 대본과 관련 콘텐츠를 자동으로 생성하는 웹 애플리케이션입니다.
- **핵심 기능:**
    - 주제 입력 및 톤/플랫폼 선택
    - AI 기반 콘텐츠 생성 (훅, 대본, 자막, 캡션, 댓글 유도)
    - 생성된 콘텐츠 복사 기능
    - 세련되고 직관적인 다크 모드 UI

## 2. 디자인 및 스타일 가이드

- **테마:** 다크 모드 (Dark Mode)
- **색상 팔레트:**
    - `배경 (dark)`: `#111015`
    - `배경 (darker)`: `#0D0C10`
    - `주요 색상 (primary)`: `#8c25f4` (보라색 계열)
- **폰트:**
    - `기본`: Noto Sans KR
    - `영문/헤드라인`: Manrope
- **레이아웃:**
    - 2단 레이아웃 (사이드바 + 메인 콘텐츠)
    - 메인 콘텐츠는 다시 2단으로 분리 (입력 섹션 + 결과 섹션)
    - 카드, 버튼 등은 둥근 모서리(rounded)와 그림자(shadow) 효과로 입체감 부여

## 3. 기능 명세 (Features)

### 3.1. 입력 섹션 (Input Section)

- **주제 입력:** 사용자가 영상의 주제나 핵심 키워드를 입력하는 `textarea`.
- **톤 선택:** 생성될 대본의 분위기를 선택하는 버튼 그룹 (예: 재미있는, 정보성, 감성적).
- **플랫폼 선택:** 영상이 게시될 플랫폼을 선택하는 버튼 그룹 (예: 쇼츠, 릴스, 틱톡).
- **생성 버튼:** 모든 옵션을 종합하여 AI에게 대본 생성을 요청하는 메인 버튼.

### 3.2. 결과 섹션 (Result Section)

- **초기 상태:** 앱의 기능과 사용법을 안내하는 UI 표시.
- **로딩 상태:** 대본 생성 요청 시, 스피너(loading animation)와 함께 생성 중임을 알리는 메시지 표시.
- **결과 표시:** 생성이 완료되면 아래 항목들을 각각의 블록으로 구분하여 표시합니다.
    - **훅 (Hook) - 3가지:** 시청자의 시선을 사로잡을 강력한 오프닝 문장 3개.
    - **30초 대본:** 영상의 흐름을 시간대별로 구분한 구체적인 대본.
    - **자막용 문장 (5개):** 영상에 빠르게 삽입할 수 있는 핵심 자막 문장 5개.
    - **캡션:** 인스타그램, 유튜브 등에 게시할 때 사용할 해시태그가 포함된 캡션.
    - **댓글 유도 문장:** 시청자의 참여를 유도하는 질문 또는 문장.
- **부가 기능:**
    - **전체 복사:** 생성된 모든 텍스트를 한 번에 복사하는 버튼.
    - **예상 소요 시간:** 생성된 대본을 읽는 데 걸리는 예상 시간 표시.
