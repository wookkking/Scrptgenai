# Blueprint: ScriptGen AI (릴스/쇼츠 스크립트 생성기)

## 1. 프로젝트 개요

- **목표:** 사용자가 입력한 주제(키워드)를 기반으로 인공지능을 활용하여 숏폼(릴스, 쇼츠, 틱톡) 비디오용 대본과 관련 콘텐츠를 자동으로 생성하는 웹 애플리케이션입니다.
- **핵심 기능:**
    - 주제 입력 및 톤/플랫폼 선택
    - AI 기반 콘텐츠 생성 (훅, 대본, 자막, 캡션, 댓글 유도)
    - 생성된 콘텐츠 복사 기능
    - 세련되고 직관적인 다크 모드 UI

## 2. 디자인 및 스타일 가이드

- **테마:** 다크 모드 (Dark Mode)
- **색상 팔레트:**
    - `배경 (dark)`: `#111015`
    - `배경 (darker)`: `#0D0C10`
    - `주요 색상 (primary)`: `#8c25f4` (보라색 계열)
- **폰트:** `Noto Sans KR`
- **레이아웃:** 2단 레이아웃 (사이드바 + 메인 콘텐츠)

## 3. 기능 명세 (Features)

### 3.1. 입력 섹션 (Input Section)
- **주제 입력:** 사용자가 영상의 주제나 핵심 키워드를 입력.
- **톤/플랫폼 선택:** 생성될 대본의 분위기와 플랫폼을 선택하는 버튼 그룹.
- **생성 버튼:** AI에게 대본 생성을 요청하는 메인 버튼.

### 3.2. 결과 섹션 (Result Section)
- **초기/로딩 상태:** 안내 메시지 또는 로딩 애니메이션 표시.
- **결과 표시:** 생성이 완료되면 아래 항목들을 각각의 **카드**로 구분하여 표시합니다.
    - **훅 (Hook) - 3가지**
    - **30초 대본**
    - **자막용 문장 (5개)**
    - **캡션**
    - **댓글 유도 문장**
- **부가 기능:**
    - **개별 복사 버튼:** 각 결과 카드 우측 상단에 해당 내용만 복사하는 버튼 추가.
    - **예상 소요 시간:** 대본 카드에 예상 시간 표시.

## 4. 현재 진행 계획 (Current Plan)

- **목표:** 결과 섹션의 구조를 개선하고, 각 항목별 '복사하기' 기능을 추가하여 사용성을 높입니다.
- **작업 단계:**
    1. **`main.js` 수정:**
        - `handleGenerateClick` 함수 내 더미 데이터 구조를 `blueprint.md`에 맞게 변경.
        - `renderResults` 함수를 수정하여, 생성된 각 콘텐츠(훅, 대본 등)를 별도의 제목과 '복사' 버튼이 포함된 카드로 렌더링하도록 변경.
        - '복사' 버튼 클릭 시, 해당 카드의 텍스트만 클립보드에 복사하는 이벤트 리스너 추가.
    2. **`style.css` 수정:**
        - 결과 섹션의 카드 레이아웃, 제목, 복사 버튼에 대한 스타일 추가.
    3. **`blueprint.md` 업데이트:** 완료 후, 다음 단계 계획 업데이트.
